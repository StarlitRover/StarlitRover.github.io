「1114」Family Property

2022/11/16 PAT（Advanced Level）Practice C＋＋ map set BFS tuple

This time, you are supposed to help us collect the data for family-owned property. Given each person’s family members, and the estate（房产）info under his/her own name, we need to know the size of each family, and the average area and number of sets of their real estate.

Input Specification:

Each input file contains one test case. For each case, the first line gives a positive integer N (≤1000). Then N lines follow, each gives the infomation of a person who owns estate in the format:

ID Father Mother k Child1​⋯Childk​ Mestate​ Area

where ID is a unique 4-digit identification number for each person; Father and Mother are the ID‘s of this person’s parents (if a parent has passed away, -1 will be given instead); k (0≤k≤5) is the number of children of this person; Childi​’s are the ID‘s of his/her children; Mestate​ is the total number of sets of the real estate under his/her name; and Area is the total area of his/her estate.

Output Specification:

For each case, first print in a line the number of families (all the people that are related directly or indirectly are considered in the same family). Then output the family info in the format:

ID M AVGsets​ AVGarea​

where ID is the smallest ID in the family; M is the total number of family members; AVGsets​ is the average number of sets of their real estate; and AVGarea​ is the average area. The average numbers must be accurate up to 3 decimal places. The families must be given in descending order of their average areas, and in ascending order of the ID’s if there is a tie.

Sample Input:

6666 5551 5552 1 7777 1 100
1234 5678 9012 1 0002 2 300
8888 -1 -1 0 1 1000
2468 0001 0004 1 2222 1 500

9012 -1 -1 3 1236 1235 1234 1 100
1235 5678 9012 0 1 50
2222 1236 2468 2 6661 6662 1 300
2333 -1 3721 3 6661 6662 6663 1 100

Sample Output:

3
8888 1 1.000 1000.000
0001 15 0.600 100.000
5551 4 0.750 100.000

打地主，分田地（bushi

给出一群人拥有的房产数和总房产面积，以及他的父母、子女信息，输出家族个数，以及每个家族的人数、平均房产数和平均面积。

本质上就是给出一个图求出连通分图的个数。我们认为在一代内有血缘关系的人之间存在一条边，即亲子关系，然后用一个邻接表存储所有边。接着对所有人进行BFS，同时flag标记是否已被访问。

🐎

#include <iostream>
#include <set>
#include <map>
#include <vector>
#include <tuple>
#include <algorithm>

using namespace std;
typedef tuple<int, int, double, double> tiidd;

struct info
{
    set<int> edge;
    int est = 0, area = 0;
    bool flag = false;
};

int main()
{
    int n, id, f, m, k, c;
    scanf("%d", &n);
    map<int, info> p;
    for (int i = 0; i < n; ++i)
    {
        scanf("%d %d %d %d", &id, &f, &m, &k);
        if (f != -1)
            p[id].edge.insert(f), p[f].edge.insert(id);
        if (m != -1)
            p[id].edge.insert(m), p[m].edge.insert(id);
        for (int j = 0; j < k; ++j)
        {
            scanf("%d", &c);
            p[id].edge.insert(c);
            p[c].edge.insert(id);
        }
        scanf("%d %d", &p[id].est, &p[id].area);
    }

    vector<tiidd> ans;
    for (auto &r: p)
    {
        if (r.second.flag) continue;
        r.second.flag = true;
        set<int> crt{r.first}, nxt;
        int sum_est = r.second.est, sum_area = r.second.area, num = 1, min_id = r.first;
        while (!crt.empty())
        {
            for (auto &t: crt)
                for (auto &z: p[t].edge)
                {
                    if (p[z].flag) continue;
                    min_id = min(min_id, z);
                    p[z].flag = true;
                    ++num, sum_est += p[z].est, sum_area += p[z].area;
                    nxt.insert(z);
                }
            crt = std::move(nxt);
            nxt = set<int>();
        }
        ans.emplace_back(min_id, num, 1.0 * sum_est / num, 1.0 * sum_area / num);
    }
    sort(ans.begin(), ans.end(),
         [](tiidd &a, tiidd &b) { return get<3>(a) == get<3>(b) ? get<0>(a) < get<0>(b) : get<3>(a) > get<3>(b); });
    printf("%zu\n", ans.size());
    for (auto &t: ans)
        printf("%04d %d %.3f %.3f\n", get<0>(t), get<1>(t), get<2>(t), get<3>(t));
}
