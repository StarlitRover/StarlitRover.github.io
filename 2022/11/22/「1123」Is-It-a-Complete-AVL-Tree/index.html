「1123」Is It a Complete AVL Tree

2022/11/22 PAT（Advanced Level）Practice C＋＋ BFS Tree AVL

An AVL tree is a self-balancing binary search tree. In an AVL tree, the heights of the two child subtrees of any node differ by at most one; if at any time they differ by more than one, rebalancing is done to restore this property. Figures 1-4 illustrate the rotation rules.

Now given a sequence of insertions, you are supposed to output the level-order traversal sequence of the resulting AVL tree, and to tell if it is a complete binary tree.

Input Specification:

Each input file contains one test case. For each case, the first line contains a positive integer N (≤ 20). Then N distinct integer keys are given in the next line. All the numbers in a line are separated by a space.

Output Specification:

For each test case, insert the keys one by one into an initially empty AVL tree. Then first print in a line the level-order traversal sequence of the resulting AVL tree. All the numbers in a line must be separated by a space, and there must be no extra space at the end of the line. Then in the next line, print YES if the tree is complete, or NO if not.

Sample Input 1:

5
88 70 61 63 65

Sample Output 1:

70 63 88 61 65
YES

Sample Input 2:

8
88 70 61 96 120 90 65 68

Sample Output 2:

88 65 96 61 70 90 120 68
NO

又见AVL！第二次看它比第一次亲切多了，但还是花了不少时间，而且是看着之前的博客码的。关于AVL旋转的详细介绍请见「1066」Root of AVL Tree。

这次需要我们输出AVL的层次遍历，同时判断是否为完全二叉树（编号与满二叉树相同）。As before，用pair对存储左右子节点，同时存储每个节点作为根节点的子树高度，都通过map进行索引。

对每一个新节点的插入，先通过递归找到插入位置，然后在递归回退的过程中找到第一个左右子树高度差>1的节点，进行AVL旋转调整同时更新相应的高度。最后BFS出层次遍历，顺便记录每个节点编号（=父节点编号*2【左子节点】/父节点编号*2+1【右子节点】），看最后一个节点编号是否=N，不等于说明不是完全二叉树。

🐎

#include <iostream>
#include <vector>
#include <map>

#define diff(root) (depth[sons[root].first]-depth[sons[root].second])
#define depth(root) (max(depth[sons[root].first],depth[sons[root].second])+1)

using namespace std;
map<int, pair<int, int>> sons;
map<int, int> depth;

void insert(int &root, int num)
{
    if (root == 0)
    {
        root = num, depth[num] = 1;
        return;
    }
    if (root > num) insert(sons[root].first, num);
    else insert(sons[root].second, num);
    depth[root] = depth(root);
    if (abs(diff(root)) < 2) return;
    bool isLeft = diff(root) > 0;
    int &so = isLeft ? sons[root].first : sons[root].second,
        &si = isLeft ? sons[root].second : sons[root].first,
        &sso = isLeft ? sons[so].first : sons[so].second,
        &ssi = isLeft ? sons[so].second : sons[so].first,
        tmp = so;
    // 判断插入节点是否位于子树的同侧
    if ((diff(root) ^ diff(so)) > 0)
    {
        so = ssi, ssi = root, root = tmp;
        depth[ssi] = depth(ssi);
        depth[root] = depth(root);
    }
    else
    {
        int &ssso = isLeft ? sons[ssi].first : sons[ssi].second,
            &sssi = isLeft ? sons[ssi].second : sons[ssi].first,
            t1 = ssso, t2 = sssi;
        ssso = so, sssi = root, so = t2, root = ssi, ssi = t1;
        // 注意高度更新顺序自下往上
        depth[ssso] = depth(ssso);
        depth[sssi] = depth(sssi);
        depth[root] = depth(root);
    }
}

int main()
{
    int n, m, r;
    cin >> n >> r;
    for (int i = 1; i < n; ++i)
    {
        cin >> m;
        insert(r, m);
    }
    bool flag = false;
    vector<pair<int, int>> crt{{r, 1}}, nxt;
    int end = -1;
    while (!crt.empty())
    {
        for (auto &k: crt)
        {
            printf("%s%d", flag ? " " : (flag = true, ""), k.first);
            if (sons[k.first].first)
                nxt.emplace_back(sons[k.first].first, 2 * k.second);
            if (sons[k.first].second)
                nxt.emplace_back(sons[k.first].second, 2 * k.second + 1);
            end = k.second;
        }
        crt = std::move(nxt);
        nxt = vector<pair<int, int>>();
    }
    printf(end == n ? "\nYES" : "\nNO");
}
