「1117」Eddington Number

2022/11/21 PAT（Advanced Level）Practice 动态规划 C＋＋

British astronomer Eddington liked to ride a bike. It is said that in order to show off his skill, he has even defined an “Eddington number”, E – that is, the maximum integer E such that it is for E days that one rides more than E miles. Eddington’s own E was 87.

Now given everyday’s distances that one rides for N days, you are supposed to find the corresponding E (≤N).

Input Specification:

Each input file contains one test case. For each case, the first line gives a positive integer , the days of continuous riding. Then N non-negative integers are given in the next line, being the riding distances of everyday.

Output Specification:

For each case, print in a line the Eddington number for these N days.

Sample Input:

10
6 7 6 9 3 10 8 2 7 8

Sample Output:

6

Ω

摘要：the maximum integer E such that it is for E days that one rides more than E miles.

给出每天的骑车里程，我们需要找到最大的E使得有E天（不必连续）的里程超过E。

这里有两个大小关系容易混淆：（里程>E的天数）≥ E。

由此我们可以得到状态转移方程：

其中代表里程>的天数，代表里程数为的天数。从而我们只要先统计出里程>N的天数和即可，然后从N～0找到第一个的天数就是答案。

🐎

#include <iostream>
#include <vector>
#include <map>

using namespace std;

int main()
{
    int n, m, k;
    cin >> n;
    map<int, int> cnt;
    vector<int> edt(n + 1, 0);
    for (int i = 0; i < n; ++i)
    {
        cin >> m;
        cnt[m] += 1;
        edt[n] += m > n;
    }
    for (k = n; k > 0 && edt[k] < k; --k)
        edt[k - 1] = edt[k] + cnt[k];
    cout << k;
}
