「1051」Pop Sequence

2022/9/1 PAT（Advanced Level）Practice C＋＋ stack

Given a stack which can keep M numbers at most. Push N numbers in the order of 1, 2, 3, …, N and pop randomly. You are supposed to tell if a given sequence of numbers is a possible pop sequence of the stack. For example, if M is 5 and N is 7, we can obtain 1, 2, 3, 4, 5, 6, 7 from the stack, but not 3, 2, 1, 7, 5, 6, 4.

Input Specification:

Each input file contains one test case. For each case, the first line contains 3 numbers (all no more than 1000): M (the maximum capacity of the stack), N (the length of push sequence), and K (the number of pop sequences to be checked). Then K lines follow, each contains a pop sequence of N numbers. All the numbers in a line are separated by a space.

Output Specification:

For each pop sequence, print in one line “YES” if it is indeed a possible pop sequence of the stack, or “NO” if not.

Sample Input:

5 7 5
1 2 3 4 5 6 7
3 2 1 7 5 6 4
7 6 5 4 3 2 1
5 6 4 3 7 2 1
1 7 6 5 4 3 2

Sample Output:

YES
NO
NO
YES
NO

Ω

难得见到的栈题，数字1-N依次入栈，给出K组出栈序列和栈的最大容量，判断它们是否可行。

首先分析出栈意味着所有比小的数字都已入过栈，由此为突破口切入。我们可以用pair来表示栈中的一串连续数字，然后我们设立一个模拟栈sim和两个变量：top——当前栈顶元素；exp——还未进栈的最小元素，接下来根据读入的数字判断是压入数字串还是出栈：

  • ：出栈

  • ：压入数字串（因为是出栈序列，所以已出栈，但考虑栈此时的容量时必须算上），更新

  • ：无需改变栈sim，但需要更新

  • ：不存在的出栈情况，如果一开始没有读入全部的出栈序列就不能直接break，因为后面还没读完


事实上，上述算法是一种逆向模拟过程，思考起来相对反人类，还有一种正向思路。由于入栈顺序肯定是1-N，那么就模拟入栈过程，将1-N逐一压栈，在压栈的过程中顺便比对出栈序列，当发现压栈数字和出栈序列当前数字吻合时，开始出栈，并将出栈序列的索引号向后移动直到栈顶元素与出栈序列当前索引元素不符，另外每次压栈时都要检查栈容量是否符合要求。最后若栈空就说明“YES”。

#include <iostream>
#include <stack>

using namespace std;

int main()
{
    int m, n, k, s;
    cin >> m >> n >> k;
    for (int i = 0; i < k; ++i)
    {
        stack<pair<int, int>> sim;
        int top = 0, exp = 1, size = 0;
        bool flag = true;
        for (int j = 0; j < n; ++j)
        {
            cin >> s;
            if (!flag) continue;
            if (top == s)
            {
                if (sim.top().first == sim.top().second)
                {
                    sim.pop();
                    top = sim.empty() ? 0 : sim.top().second;
                }
                else
                    sim.top().second = (--top);
                size -= 1;
            }
            else if (s > exp)
            {
                sim.emplace(exp, s - 1);
                if ((size + s - exp + 1) > m)
                    flag = false;
                else size += (s - exp);
                top = s - 1;
                exp = s + 1;
            }
            else if (s == exp)
                exp += 1;
            else
                flag = false;
        }
        cout << (flag ? "YES" : "NO") << endl;
    }
}

#include <iostream>
#include <vector>
#include <stack>

using namespace std;

int main()
{
    int m, n, k;
    cin >> m >> n >> k;
    for (int i = 0; i < k; ++i)
    {
        vector<int> seq(n);
        stack<int> sim;
        for (auto &s: seq)
            cin >> s;
        int idx = 0;
        for (int j = 0; j < n; ++j)
        {
            sim.push(j + 1);
            if (sim.size() > m) break;
            while (!sim.empty() && sim.top() == seq[idx])
            {
                sim.pop();
                ++idx;
            }
        }
        cout << (sim.empty() ? "YES" : "NO") << endl;
    }
}
