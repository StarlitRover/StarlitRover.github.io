「1013」Battle Over Cities

2022/3/12 PAT（Advanced Level）Practice C＋＋ set BFS

➳ENTRY

It is vitally important to have all the cities connected by highways in a war. If a city is occupied by the enemy, all the highways from/toward that city are closed. We must know immediately if we need to repair any other highways to keep the rest of the cities connected. Given the map of cities which have all the remaining highways marked, you are supposed to tell the number of highways need to be repaired, quickly.

For example, if we have 3 cities and 2 highways connecting and . Then if is occupied by the enemy, we must have 1 highway repaired, that is the highway .

Input Specification:

Each input file contains one test case. Each case starts with a line containing 3 numbers N (<1000), M and K, which are the total number of cities, the number of remaining highways, and the number of cities to be checked, respectively. Then M lines follow, each describes a highway by 2 integers, which are the numbers of the cities the highway connects. The cities are numbered from 1 to N. Finally there is a line containing K numbers, which represent the cities we concern.

Output Specification:

For each of the K cities, output in a line the number of highways need to be repaired if that city is lost.

Sample Input:

3 2 3
1 2
1 3
1 2 3

Sample Output:

1
0
0

Ω

首先极简翻译一波。简单的说就是，有一些城市，部分城市之间连有高速公路。当一个城市被敌人占领后，与其相连的高速公路都无法通行，但要求其余城市必须是连通的，试问最少需要搭几条高速公路。

乍看题目有点迷，其实只需要求剩余城市的连通分图数即可。若剩余城市含个连通分图，那么显然最少只需条边即可将所有剩余城市相连。

我采取的是Alg. BFS（广度优先搜索）的思想，按顺序遍历每一座城市，然后将与这座城市相连的城市进行标记，接着再将与这些城市相连的城市进行标记，逐层递归，直至相连的城市数量为0。当然对于被敌人占领的那一座城市我们需要在一开始对其就进行标记，从而后面递归的时候就无法从这座城市经过。

C☺DE

#include <iostream>
#include <map>
#include <set>
#include <vector>

using namespace std;

int main()
{
    int n, m, k, a, b;
    cin >> n >> m >> k;
  //记录高速公路
    map<int, vector<int>> connect;
    for (int i = 0; i < m; ++i)
    {
        cin >> a >> b;
        connect[a].push_back(b);
        connect[b].push_back(a);
    }
  //标记变量
    vector<bool> mark(n + 1);
    int city, num;
    for (int i = 0; i < k; ++i)
    {
        cin >> city;
        num = 0;
        fill(mark.begin(), mark.end(), false);
        mark[city] = true;
        for (int j = 1; j <= n; ++j)
        {
            if (mark[j])
                continue;
          //mark[j]=false说明是一个新的连通分图
            num += 1;
          //cup存储与next中城市相连的城市
            set<int> next{j}, cup;
            while (!next.empty())
            {
                for (auto &c: next)
                    if (mark[c])
                        continue;
                    else
                    {
                        mark[c] = true;
                        cup.insert(connect[c].begin(), connect[c].end());
                    }
                next = cup;
                cup.clear();
            }
        }
        cout << num - 1 << endl;
    }
}

⚡︎ 算法说明

  1. 使用map可以直接通过城市序号访问与其相连的城市vector
  2. fill(mark.begin(), mark.end(), false);将mark迭代器中的值全部填充为false
  3. 用set记录与当前城市相连的城市可以避免重复
  4. cup.insert(connect[c].begin(), connect[c].end());将vector<int> connect[c]中的值插入到set<int> cup的尾部
  5. 本题也可以采用Alg. DFS（深度优先搜索），详情可见柳婼 の blog
