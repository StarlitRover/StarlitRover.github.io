『1167」DCartesian Tree

2022/12/2 PAT（Advanced Level）Practice C＋＋ Tree

A Cartesian tree is a binary tree constructed from a sequence of distinct numbers. The tree is heap-ordered, and an inorder traversal returns the original sequence. For example, given the sequence { 8, 15, 3, 4, 1, 5, 12, 10, 18, 6 }, the min-heap Cartesian tree is shown by the figure.

Your job is to output the level-order traversal sequence of the min-heap Cartesian tree.

Input Specification:

Each input file contains one test case. Each case starts from giving a positive integer N (≤30), and then N distinct numbers in the next line, separated by a space. All the numbers are in the range of int.

Output Specification:

For each test case, print in a line the level-order traversal sequence of the min-heap Cartesian tree. All the numbers in a line must be separated by exactly one space, and there must be no extra space at the beginning or the end of the line.

Sample Input:

10
8 15 3 4 1 5 12 10 18 6

Sample Output:

1 3 5 8 4 6 15 10 12 18

笛卡尔树是通过一串序列进行构造的，它满足最小堆的特征（父节点<子节点），同时它的中序遍历结果就是构造序列。

由于构造序列与中序遍历是一致的，那么在逐一插入节点的过程中，我们只能将新节点插入到现有节点的右边（原有节点是新节点的左子树/新节点是原有节点的右子树），才能保证中序遍历中新节点在现有节点的后面。然后最小堆的特征约束了小节点必须在大节点的上方，可以认为一个约束了左右，另一个约束了上下，那么就可以制定如下插入方案：

  1. 新节点>当前节点：向右子树插入

  2. 新节点<当前节点：将当前节点作为新节点的左子树

每次都从根节点开始插入，递归插入函数返回插入后子树的根节点。另外由于节点键值是整型范围，因此还需要一个bool数组标记各个节点的子节点是否存在。

#include <iostream>
#include <map>
#include <vector>

using namespace std;
map<int, pair<int, int>> sons;
map<int, pair<bool, bool>> has;

int insert(int r, int k)
{
    if (r < k && has[r].second)
        sons[r].second = insert(sons[r].second, k);
    else if (r < k)
        sons[r].second = k, has[r].second = true;
    else if (r > k)
    {
        sons[k].first = r, has[k].first = true;
        return k;
    }
    return r;
}

int main()
{
    int n, r, k;
    cin >> n >> r;
    for (int i = 1; i < n; ++i)
        cin >> k, r = insert(r, k);
    vector<int> crt{r}, nxt;
    bool space = false;
    while (!crt.empty())
    {
        for (auto &t: crt)
        {
            printf("%s%d", space ? " " : (space = true, ""), t);
            if (has[t].first) nxt.push_back(sons[t].first);
            if (has[t].second) nxt.push_back(sons[t].second);
        }
        crt = std::move(nxt), nxt = vector<int>();
    }
}
