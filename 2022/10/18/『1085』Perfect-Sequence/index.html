『1085』Perfect Sequence

2022/10/18 PAT（Advanced Level）Practice C＋＋ sort vector

Given a sequence of positive integers and another positive integer p. The sequence is said to be a perfect sequence if M≤m×p where M and m are the maximum and minimum numbers in the sequence, respectively.

Now given a sequence and a parameter p, you are supposed to find from the sequence as many numbers as possible to form a perfect subsequence.

Input Specification:

Each input file contains one test case. For each case, the first line contains two positive integers N and p, where  is the number of integers in the sequence, and  is the parameter. In the second line there are N positive integers, each is no greater than .

Output Specification:

For each test case, print in one line the maximum number of integers that can be chosen to form a perfect subsequence.

Sample Input:

10 8
2 3 20 4 5 1 6 7 8 9

Sample Output:

8

Ω

完美序列指的是满足的序列，给出一组序列和，输出最多能够从这个序列中抽出元素构成完美序列的个数。

很淳朴的思想，设定一个只加不减的偏移量（1个数字必然成立），先sort数组然后顺序遍历，判断是否成立，成立则继续找到上限，否则continue。

最后case 5挂了，检查后发现PTA的clang和g++都是32位的，即int的长度是long int的一半，最大为，目前64位电脑主流编译器上的int基本都是32位。因此将p改成long int即可。

🐎

#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main()
{
    long int n, p;
    cin >> n >> p;
    vector<int> arr(n);
    for (auto &a: arr)
        cin >> a;
    sort(arr.begin(), arr.end());
    int ans = 1;
    for (int i = 0; i < arr.size() - ans; ++i)
        while (i + ans < arr.size() && arr[i + ans] <= arr[i] * p)
            ++ans;
    cout << ans;
}
