「1100」Mars Numbers

2022/10/29 PAT（Advanced Level）Practice C＋＋ string

People on Mars count their numbers with base 13:

  • Zero on Earth is called “tret” on Mars.
  • The numbers 1 to 12 on Earth is called “jan, feb, mar, apr, may, jun, jly, aug, sep, oct, nov, dec” on Mars, respectively.
  • For the next higher digit, Mars people name the 12 numbers as “tam, hel, maa, huh, tou, kes, hei, elo, syy, lok, mer, jou”, respectively.

For examples, the number 29 on Earth is called “hel mar” on Mars; and “elo nov” on Mars corresponds to 115 on Earth. In order to help communication between people from these two planets, you are supposed to write a program for mutual translation between Earth and Mars number systems.

Input Specification:

Each input file contains one test case. For each case, the first line contains a positive integer N (<100). Then N lines follow, each contains a number in [0, 169), given either in the form of an Earth number, or that of Mars.

Output Specification:

For each number, print in a line the corresponding number in the other language.

Sample Input:

4
29
5
elo nov
tam

Sample Output:

hel mar
may
115
13

Ω

火星上面不用字母，谢谢。

给地球和火星翻译数字，假设火星上的数字是13进制的，而且最多两位，而且每一位的数字表示都不同，另有以下规则：

  1. 0需要单独输出“tret”

  2. 某位上的数字为0时不输出

  3. 如果两位都输出中间必须有空格

真的很讨厌这种分类讨论题，一不小心就把所有坑都踩了一遍。首先根据读入的一行字符串第一个字符是否为数字来判断是火星数字还是地球数字。

如果是地球数字，分别对13进行除和取余操作，空格需要在两者都不为0时出现，⚠️0需单独输出；如果是火星数字，判断是否含空格，有空格就分别查找对应的数字，无空格时要对两位上的火星数字表示都查找一遍，⚠️“tret”。

🐎

#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main()
{
    vector<string> high{"", "tam", "hel", "maa", "huh", "tou", "kes", "hei", "elo", "syy", "lok", "mer", "jou"},
                   low{"", "jan", "feb", "mar", "apr", "may", "jun", "jly", "aug", "sep", "oct", "nov", "dec"};
    int n;
    cin >> n;
    getchar();
    string s;
    for (int i = 0; i < n; ++i)
    {
        getline(cin, s);
        if (isdigit(s[0]))
        {
            int k = stoi(s);
            if (k) cout << high[k / 13] << (k % 13 && k / 13 ? " " : "") << low[k % 13] << endl;
            else cout << "tret\n";
        }
        else
        {
            int space = s.find(' ');
            if (space == string::npos)
            {
                if (s == "tret") cout << "0\n";
                else
                {
                    int k = int(find(low.begin(), low.end(), s) - low.begin());
                    cout << (k == low.size() ? 13 * (find(high.begin(), high.end(), s) - high.begin()) : k) << endl;
                }
            }
            else
            {
                int a = int(find(high.begin(), high.end(), s.substr(0, space)) - high.begin()),
                        b = int(find(low.begin(), low.end(), s.substr(space + 1, s.size() - space - 1)) - low.begin());
                cout << a * 13 + b << endl;
            }
        }
    }
}
